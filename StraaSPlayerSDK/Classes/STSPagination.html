STSPagination

@interface STSPagination : NSObject

Pagination model of StraaS player SDK STSPagination is an object wrapping pagination header in api request. reference: https://developer.github.com/guides/traversing-with-pagination/

  • last

    The number of the last page. Usually, last page is a constant, unless CMS update resource. Nil if at the last page.
