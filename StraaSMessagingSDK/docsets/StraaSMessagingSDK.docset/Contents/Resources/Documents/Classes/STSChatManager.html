    Connect to chat room with user JWT.

    Declaration

    Objective-C

    - (void)connectToChannel:(nonnull NSString *)channelCode
                     JWT:(nonnull NSString *)JWT
              autoCreate:(BOOL)autoCreate
           eventDelegate:(nullable id<STSChatEventDelegate>)eventDelegate;

    Swift

    func connectToChannel(channelCode: String, JWT: String, autoCreate: Bool, eventDelegate: STSChatEventDelegate?)

    Parameters

    channelCode

    Identifier of chat room, channel code and chat room is a 1-to-1 mapping. Don’t pass nil or this method return early.
    JWT

    User identity infromation. For StraaS.io CMS member, a member JWT should be used here. Use an empty string to represent guest. Don’t pass nil or this method return early.
    autoCreate

    Create a new chat room or not, if the channel code can not be found.
    eventDelegate

    Chat event delegate. Chat manager not retain this delegate.
